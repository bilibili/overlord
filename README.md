Overlord is a proxy based high performance Memcached&Redis solution written in Go.

# Features

- [ ] 支持Redis & Memcache 协议
- [ ] 提供连接池让所有client共享，减少对cache的连接个
- [ ] 多重散列支持
- [ ] 前戳路由
- [ ] cache备份
- [ ] 心跳 & 故障转移
- [ ] 支持“温热”缓存
- [ ] 广播操作
- [ ] 热加载
- [ ] 多级缓存
- [ ] 服务质量保证（限流、限速）
